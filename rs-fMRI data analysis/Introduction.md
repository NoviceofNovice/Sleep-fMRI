# rs-fMRI data analysis

rs-fMRI를 분석하는 방법에는 여러가지 방법론들이 존재한다.

아래는 존재하는 방법론들이다.

* Seed-based functional connectivity analysis
* ICA
* ReHo analysis
* fALFF analysis
